# LaraFy CMS: Plataforma de Blogging Monolítica

## ¿Qué es LaraFy CMS?
LaraFy CMS es un sistema de gestión de contenidos (CMS) moderno y potente, diseñado específicamente para blogging. Construido sobre la base de **Laravel 12**, este proyecto se enfoca en ofrecer una experiencia de usuario "premium" y una arquitectura robusta, siguiendo las mejores prácticas de desarrollo web contemporáneo.

## Características Clave
- **Gestión Integral de Publicaciones**: Sistema CRUD completo para noticias y artículos.
- **Control de Acceso (RBAC)**: Roles definidos de `Admin` y `Author` para una gestión segura de contenidos.
- **Diseño Premium**: Interfaz moderna con *Glassmorphism*, gradientes animados y tipografía optimizada (Inter).
- **Localización Completa**: Soporte nativo para el idioma **Español**, incluyendo fechas y mensajes de validación.
- **Visuales Dinámicos**: Soporte para imágenes destacadas con generación automática vía `picsum.photos` como respaldo.
- **Arquitectura Monolítica**: Simplifica el despliegue y desarrollo manteniendo un alto rendimiento.

## Stack Tecnológico
- **Backend**: Laravel 12 (PHP 8.2+)
- **Base de Datos**: PostgreSQL
- **Frontend**: Blade, Tailwind CSS 4, Alpine.js
- **Testing**: PestPHP
- **Estilos**: @tailwindcss/typography para un renderizado de texto impecable.
